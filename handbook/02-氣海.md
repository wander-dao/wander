# 卷二 · 氣海與寒月

🌐 [English](en/02-qi-sea.md)

## 器之涯

氣海初成,容一千萬靈氣;每進一層,約增五分之一,九層而五倍。境進,則器宏。

未滿之時,新氣靜蓄池中,不自化物。

## 寒月

池滿,方觸自動之法;法若為 off,溢者散於寒月,不復得——累計見於行囊。唯[重算](04-境界.md)按全史重計,散者盡贖。

## 自動之法

```bash
wander config set autoMode cultivate   # 滿則自煉,入修為
wander config set autoMode compress    # 滿則自轉,成靈石
wander config set autoMode off         # 默認:溢而散
```

| 法 | 滿時 | 寒月 |
|----|------|------|
| off | 納一次,餘者散 | 有 |
| cultivate | 滿即煉,循環納盡 | 無 |
| compress | 滿即轉,循環納盡 | 唯不足一下品之餘 |

## 行法

- 志在境界,不欲勞心——cultivate。
- 為他日靈石之用囤資——compress。
- 存手動抉擇之趣——off,而候 statusline 之警(⚠ 寒月+X,見[卷五](05-觀測.md))。
