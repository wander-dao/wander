# 附錄甲 · 指令

🌐 [English](en/06-commands.md)

## 行法

| 令 | 效 |
|----|-----|
| `wander bag` | 行囊:修為 / 氣海 / 靈石 / 寒月 / 靈根 |
| `wander cultivate [靈氣]` | 煉化(默認盡池)→ 修為 |
| `wander compress [靈氣]` | 轉化(默認盡池)→ 靈石 |
| `wander decompress <下品數>` | 吸收靈石回池 |
| `wander reset [--to cultivation\|stones]` | 全史重算(見[卷四](04-境界.md)) |

## 觀之

`wander stats` 現全景;一旗一景:

| 旗 | 景 |
|----|-----|
| `--today` / `--week` / `--month` / `--all` | 今日 / 本週 / 本月 / 全歷 |
| `--sessions` | 歷練榜 |
| `--tools` | 法寶譜(內建與靈像同榜) |
| `--skills` | 術法圖鑑,記習得之日——唯記模型自開之卷 |
| `--peaks` | 生平之最,與吐納 |
| `--streak` | 修行日曆 |
| `--realm` | 境界詳情 |
| `--projects` | 諸洞府 |
| `--oracle` | 今日籤,皆真跡 |
| `--essence` | 真元(新算)與藏氣(既存調閱)之分,逐時段與逐洞府 |
| `--models` | 各 model 用量與薪火(fable / opus / 自訂 各自成列,按薪火高低;預設全歷) |
| `--share` | 分享模式:洞府、外接法寶與自訂術法一律化名(丹霞、漱石之屬),真實路徑與名字不現——截圖安心曬。可疊加於任何視圖 |
| `--json` | 機讀,純觀 |

修飾:`--refresh` 強制重掃;`--cwd <path>` 指洞府;`-g`/`--global` 看全歷、`-f`/`--focus` 聚焦當前洞府(施於 `--models`/`--tools`/`--skills`;`--models` 本就全歷,`--tools`/`--skills` 本就聚焦)。

> 真元約當 Claude Code `/stats` 所示之 token 總量;靈氣兼計藏氣(既存反覆調閱),故數十倍於之——二者所量不同,非有誤。

## 所設

| 令 | 效 |
|----|-----|
| `wander config list` | 覽所設 |
| `wander config set project.<絕對路徑>.alias "名"` | 洞府別名 |
| `wander config set tool.<名>.alias "名"` | 法寶改名 |
| `wander config set skill.<名>.alias "名"` | 術法改名 |
| `wander config set mcp.<server>.alias "名"` | 靈像整尊改名 |
| `wander config set autoMode <off\|cultivate\|compress>` | 池滿之法(見[卷二](02-氣海.md)) |
| `wander config prices` | 覽薪火現價:官定家族價 + 你所訂 |
| `wander config set price.<model> <入>/<出>[/<寫>/<讀>]` | 自訂模型單價(USD / 1M token) |
| `wander config set price.default <入>/<出>` | 未知模型之價 |
| `wander config set price.<model> <入>/<出> --from YYYY-MM-DD` | 分段定價,自該日生效 |
| `wander config get <鍵>` | 讀一鍵 |
| `wander config unset <鍵>` | 除一設(支援 set 的所有鍵;`price.<model> --from <日>` 可只除一段) |
| `wander config reset` | 盡復其初;年譜不動 |

價,即改即生:薪火讀時重計,史不改。省快取價者,按官比推——寫 1.25 倍入價,讀 0.1 倍。

## 補全

shell 補全(zsh / bash / fish):

```bash
source <(wander completions zsh)    # 或寫入 ~/.zshrc 永續
```

## 更新

```bash
wander version               # 版與臺(--version 只出版號)
wander upgrade               # 升至最新
wander upgrade v0.1.0        # 指版而裝
wander upgrade --check       # 只問新舊,不取
```

純手動,不自行更新,不錄行蹤:唯行令時方叩 GitHub 取新版,分毫不出。取而 **核驗無誤方換**;驗敗即止,舊者不動。

`upgrade` 唯進。欲返舊版,以 installer 指版:

```bash
curl -fsSL https://raw.githubusercontent.com/wander-dao/wander/main/install.sh | bash -s -- --version v0.1.0
```

若資料出自更新之版,舊版首讀,必先問而後行;返版之前,宜備份永久一層。

## 解除安裝

```bash
wander uninstall                 # 相問而行;默認留年譜與所設
wander uninstall --purge-config  # 並清所設
wander uninstall --purge-archive # 並焚年譜(不可再生,再問一次)
wander uninstall --purge         # 二者並清
wander uninstall --clean-statusline  # 並除 settings.json 之接線
wander uninstall --dry-run       # 先觀其計
```

默認**年譜與所設俱留**,再裝可續。欲重來而不失史,行[重算](04-境界.md)可也。
