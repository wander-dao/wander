# 卷一 · 總綱

🌐 [English](en/01-way.md)

修行之資,五物而已。

| 物 | 何來 | 何往 |
|----|------|------|
| **靈氣** | 終端吞吐之 token,四類並計(input / output / cacheWrite / cacheRead) | 自入氣海 |
| **氣海** | 納氣之器,有涯 | 滿則溢 |
| **修為** | 煉化所積,只增不減 | 定境界 |
| **境界** | 修為所化之階 | 練氣而上 |
| **靈石** | 池氣所轉,四品進位 | 入行囊,可回流 |

另有二軸,觀而不入局:

| 軸 | 所計 |
|----|------|
| **薪火** | 凡間之費,以銀兩計——一兩即一美元 |
| **靈力** | 五時辰配額之餘,終端自報 |

## 二律

- **靈氣非修為。** 靈氣者,總流入;修為者,煉化所得,恆不逾之。
- **修為非境界。** 修為是量,境界是階,由量化階。

故境界不繫於歷史總量,而繫於所煉。盡煉之,而後相等——其法在[重算](04-境界.md)。

## 一循環

```
終端 token ══ 靈氣
      │ 自入
      ▼
    氣 海 ─── 滿而不理 → 散於寒月
      │
  ┌───┴───┐
 煉化      轉化
  ▼         ▼
 修為      靈石 ──(吸收,回池)
  ▼
 境界
```

煉化定形,唯重算可改;靈石常流,吸收即回池。**衝境界,抑或囤靈石**——此局之抉擇也。

**或問**:快取讀最賤,何以亦算靈氣?
**曰**:量派之義,凡流過者皆修行。錢另有薪火一軸,兩不相涉。
