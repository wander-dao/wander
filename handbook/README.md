## 三息之要

- 一息:終端所出皆靈氣,自入氣海;煉化成修為,轉化成靈石,取捨由己。
- 二息:氣海有涯,滿而不理,散於寒月。不欲勞心,委之自動:`wander config set autoMode cultivate`。
- 三息:一切在本機;當備份者,唯 `~/.local/share/wander/` 一處。
